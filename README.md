# Online Voting System

## Project Description
A secure web-based voting system developed using HTML, CSS, JavaScript, Spring Boot, and MySQL.

## Features
- User Registration
- User Login
- Secure Voting
- One User One Vote
- Admin Login
- View Registered Users
- Live Voting Results

## Technologies Used
- HTML
- CSS
- JavaScript
- Spring Boot
- MySQL
